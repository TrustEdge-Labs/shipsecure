IMPORTANT: Use individual named imports only (NOT barrel import). This enables tree-shaking so only these 4 icons are bundled.

Key details:
- Remove `text-xl` from the parent div (no longer needed — emoji sizing class, SVG uses explicit w-6 h-6)
- Keep `text-brand-primary` on parent div (icons inherit this color via currentColor)
- Use `w-6 h-6` (24px) for consistent sizing matching Lucide's default
- Use `aria-hidden="true"` because these are decorative (adjacent h3 text provides semantic meaning)
- Do NOT add aria-label — these icons have text labels next to them, aria-label would be redundant
- Do NOT add "use client" directive — Lucide React icons are simple SVG components that work in server components with Next.js 16
- Do NOT use the `color` prop — let currentColor inherit from parent's text-brand-primary

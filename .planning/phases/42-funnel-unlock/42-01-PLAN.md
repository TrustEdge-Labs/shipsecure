**In src/db/scans.rs — add two new query functions:**

1. `count_scans_by_domain_last_hour(pool: &PgPool, domain: &str) -> Result<i64, sqlx::Error>`:
   - SQL: `SELECT COUNT(*) FROM scans WHERE target_url LIKE $1 AND created_at >= NOW() - INTERVAL '1 hour'`
   - Bind pattern: `format!("%://{domain}%")` (same pattern as `count_anonymous_scans_by_email_and_domain_today`)
   - Counts ALL scans (anonymous + authenticated) for the domain in the last hour

2. `get_recent_completed_scan_for_domain(pool: &PgPool, domain: &str) -> Result<Option<Scan>, sqlx::Error>`:
   - SQL: `SELECT [full Scan columns] FROM scans WHERE target_url LIKE $1 AND status = 'completed' AND (expires_at IS NULL OR expires_at > NOW()) ORDER BY completed_at DESC LIMIT 1`
   - Bind pattern: `format!("%://{domain}%")`
   - Returns the most recent non-expired completed scan for the domain (any tier, any user)

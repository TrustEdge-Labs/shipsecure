Add the `pub fn parse(content: &str) -> Result<Vec<ParsedDep>, SupplyChainError>` function and its helpers to `lockfile_parser.rs`.

**Main parse function:** Parse JSON, read `lockfileVersion` field (default to 1 if absent), dispatch:
- lockfileVersion 3 -> `parse_v3()`
- lockfileVersion 1 or 2 -> check if `packages` key exists; if yes (v2 with packages), use `parse_v3()` since v2's packages key has same structure as v3. If only `dependencies` key exists, use `parse_v1_v2_nested()`.
- Any other version -> return `Err(SupplyChainError::LockfileParse(format!("Unsupported lockfileVersion: {}", version)))`

**parse_v3 function (handles v3 and v2-with-packages):**
- Get `packages` as Object, error if missing
- Iterate entries. Skip key `""` (root entry).
- Extract package name: strip everything up to and including the LAST `node_modules/` occurrence in the key. This handles both `node_modules/@babel/core` and `node_modules/parent/node_modules/@babel/core` correctly.
- Read `version` as string. Skip entries with empty or missing version.
- Dedup using `HashSet<(String, String)>` on (name, version). Skip if already seen.
- Classify source per D-05 using `classify_source()` helper.
- Read `dev` as bool (default false). Set `is_dev`.
- Push `ParsedDep`.

**parse_v1_v2_nested function (handles v1 and v2-without-packages):**
- Get `dependencies` as Object, error if missing
- Recursively walk: for each entry, the key IS the package name. Read `version`, `resolved`, `dev` fields.
- If entry has nested `dependencies` object, recurse into it.
- Dedup using same HashSet<(String, String)> pattern (passed as mutable ref through recursion).
- Classify source, set is_dev.

**classify_source helper:**
```rust
fn classify_source(pkg: &serde_json::Value) -> DepSource {
    match pkg.get("resolved").and_then(|r| r.as_str()) {
        Some(r) if r.starts_with("git+") => DepSource::Git,
        Some(r) if r.starts_with("file:") => DepSource::File,
        Some(r) if r.starts_with("link:") => DepSource::Link,
        Some(_) => DepSource::Registry,  // has a resolved URL (registry tarball)
        None => DepSource::Tarball,       // no resolved field
    }
}
```

Also check for `"link": true` field (workspace packages) which should be classified as DepSource::Link even if resolved is absent.

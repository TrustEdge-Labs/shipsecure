**2. Update the make_span_with closure** to store RequestId in request extensions:

The research notes (Pattern 3 alternative) recommend storing request_id in request extensions during TraceLayer creation for cleaner extraction in handlers. However, `make_span_with` receives `&Request` (immutable reference), so we cannot insert into extensions there.

**Instead, use a middleware layer to insert the RequestId extension BEFORE TraceLayer:**

Add an `axum::middleware::from_fn` layer that runs before TraceLayer. This middleware:
1. Generates UUID v4
2. Inserts `RequestId(uuid)` into request extensions
3. Passes through to TraceLayer (which reads from extensions for the span)

```rust
use axum::middleware;

async fn inject_request_id(
    mut request: axum::extract::Request,
    next: axum::middleware::Next,
) -> axum::response::Response {
    let request_id = uuid::Uuid::new_v4();
    request.extensions_mut().insert(RequestId(request_id));
    next.run(request).await
}
```

Then update the make_span_with closure to READ the request_id from extensions instead of generating it:

```rust
.make_span_with(|request: &Request<axum::body::Body>| {
    let request_id = request.extensions()
        .get::<RequestId>()
        .map(|r| r.0)
        .unwrap_or_else(uuid::Uuid::new_v4);  // Fallback if extension missing

    tracing::info_span!(
        "http_request",
        request_id = %request_id,
        method = %request.method(),
        uri = %request.uri().path(),
        status_code = tracing::field::Empty,
        latency_ms = tracing::field::Empty,
    )
})
```

**3. Update the Router middleware ordering:**

Middleware layers execute bottom-to-top with `.layer()`. We need:
1. inject_request_id runs FIRST (outermost)
2. trace_layer runs SECOND (sees RequestId extension)
3. cors runs THIRD

```rust
let app = Router::new()
    // API routes
    .route("/api/v1/scans", post(scans::create_scan))
    // ... other routes ...
    .layer(cors)
    .layer(trace_layer)
    .layer(middleware::from_fn(inject_request_id))
    .with_state(state)
    // Health checks bypass all middleware
    .route("/health", get(|| async { "ok" }))
    .into_make_service_with_connect_info::<SocketAddr>();
```

With `.layer()` bottom-to-top: inject_request_id (bottom) → trace_layer (middle) → cors (top). Request flows: inject_request_id → trace_layer → cors → handler.

**Inline helper functions (defined inside the file, not exported):**

- `SeverityBadge({ count, level })` — renders colored number chip. Colors: critical = `bg-danger-bg text-danger-text border-danger-border`, high = `bg-caution-bg text-caution-text border-caution-border`, medium = `bg-info-bg text-info-text border-info-border`, low = `bg-success-bg text-success-text border-success-border`. Returns `null` if count is 0.

- `TierBadge({ tier })` — renders "Enhanced" (for tier === 'authenticated') with `bg-brand-primary/10 text-brand-primary border-brand-primary/20` or "Basic" with `bg-surface-secondary text-text-secondary border-border-subtle`. Uses `rounded-full` pill shape, `text-xs font-medium`.

- `formatExpiry(expiresAt, status)` — returns React node:
  - If status is 'failed': red "Failed" badge
  - If expiresAt is null: em dash "—"
  - If expired (days <= 0): gray "Expired" badge
  - If days <= 3: caution-colored "{N} day(s) left"
  - Otherwise: secondary-colored "{N} days left"

- `formatDate(dateStr)` — returns formatted date string using `toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric' })`.

- `extractHostname(targetUrl)` — extracts hostname from full URL using `new URL(targetUrl).hostname`, with try/catch fallback to raw string.

- `Pagination({ currentPage, totalPages })` — renders numbered page links using `<Link href={'/dashboard?page=${n}'}>`; returns null if totalPages <= 1. Current page is non-clickable bold text; other pages are links. Use `inline-flex items-center gap-1` layout.

**Table layout (visible `sm:` and up):**
- Wrapped in `<div className="hidden sm:block">`
- `<table>` with columns: Domain, Date, Severity, Expiry, Tier, Action
- Severity column renders all four `SeverityBadge` components inline with `flex gap-1`
- For non-expired, non-failed scans: entire row wrapped in `<Link>` to `/results/${scan.results_token}`, action column has explicit "View" button (also a Link)
- For expired scans: row has `opacity-60` class, no Link wrapper, action column shows "Expired" badge instead of View button
- For failed scans: row has normal opacity, no Link wrapper, action column shows "Failed" badge
- Row hover: `hover:bg-surface-secondary` transition for non-expired/non-failed rows

**Mobile card layout (visible below `sm:`):**
- Wrapped in `<div className="sm:hidden space-y-3">`
- Each scan renders as a card with `border border-border-subtle rounded-lg p-4 bg-surface-elevated`
- Card shows: hostname (bold), date, severity badges row, expiry text, tier badge
- Non-expired cards wrapped in Link; expired cards dimmed with opacity-60
